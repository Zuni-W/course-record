# 总述

本项目分别实现了以下三个部分：

1. 基于国家密码管理局SM2椭圆曲线公钥密码算法描述的SM2加解密算法

2. 基于国家密码管理局SM2椭圆曲线公钥密码算法描述的SM2签名验证算法

3. 基于SM2的PGP协议

4. 基于SM2的环签名算法

# SM2加解密算法

由于时间限制，仅考虑使用推荐参数的情况
在程序中，我们对比特流使用了'01'字符串模拟的方法,字节串使用十六进制字符串模拟
内部的哈希函数使用了自主实现的SM3
